# 🌐 My Personal Website

Welcome to the source code for my personal corner of the internet! This project is a clean, modern, and responsive "About Me" page designed to showcase my skills and provide a way for people to get in touch.

## 🚀 Features
- **Responsive Design:** Optimized for desktops, tablets, and mobile phones.
- **Modern UI:** A professional color palette using a clean, minimalist aesthetic.
- **Quick Links:** Easy-to-find contact buttons and skill tags.
- **SEO Ready:** Structured with semantic HTML5 for better search engine visibility.

## 🛠️ Built With
- **HTML5:** For the structure and content.
- **CSS3:** For the styling and layout (no external frameworks required!).
- **Gemini AI:** Assisted in the initial design and content strategy.
